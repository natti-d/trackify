# Trackify
HTML, CSS and JavaScript website (multipage) which is assigned for Diploma project in High School, last year of "Acad. Ivan Gyuzelev" High School of Natural Sciences and Mathematics, Gabrovo.<br>
Topic of project: Development of a web application for project development management and analysis.<br>
The project is developed with the help of Canva, Visual Studio Code, dbdiagram.io and XAMPP Portable.<br>
The technologies used are HTML5, CSS3, Bootstrap 5.3.2, JavaScript, MySQL and PHP. Every resource is copyrighted.<br>
The whole project is placed in the master branch.
